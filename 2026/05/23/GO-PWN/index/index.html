GO-PWN

接下来准备学习 gopwn，毕竟 23,24 年 CISCN 连续出了两道 go 类型的 pwn，那首先就要了解 go 的基本语法

第一个程序：Hello World

在 Go 中，任何可执行程序都必须包含一个 main 包和一个 main 函数。

注意点：

  • Go 语句末尾不需要分号（除非多条语句写在一行）。
  • 左大括号 {必须和函数声明在同一行，不能另起一行（这是 Go 的硬性规定）。

2. 变量与常量 (Variables & Constants)

Go 是强类型语言，但它有非常聪明的类型推断功能。

3. 基本数据类型

  • 布尔型:bool (true 或 false)
  • 字符串:string (用双引号 “” 包裹)
  • 整型:int, int8, int32, int64 等 (通常直接用 int)
  • 浮点型:float32, float64 (通常使用 float64 以保证精度)

4. 控制结构 (Control Structures)

Go 的控制结构抛弃了多余的括号，看起来非常清爽。

If-Else 条件判断

条件判断不需要小括号()，但大括号 {} 是必须的。

For 循环 (Go 只有 for 循环)

Go 语言中没有 while 或 do-while 关键字，所有的循环都由 for 来承担。

5. 函数 (Functions)

函数的关键字是 func。Go 语言的函数有一个非常强大的特性：支持返回多个值。

总结：

  1. 用 package main 和 func main() 启动程序
  2. 定义变量最常用短声明 :=
  3. if 和 for 的条件不需要加括号 ()
  4. 函数可以有多个返回值。
